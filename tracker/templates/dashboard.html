Simple Email Tracking

Create a tracker. Paste it into your email. Watch opens appear.

This page wraps the raw API in a simpler flow. You create a tracking pixel, copy the generated HTML snippet, and paste it into an HTML email or email signature. When the image loads, the open is recorded here.

1. Create a tracker Give it a clear name like "March follow-up" or "Client proposal".
2. Copy the email snippet Paste the generated image tag into the HTML version of your email.
3. Check opens below Each image load adds a new open entry with time, IP, and user agent.

Create a Tracker

Start with a simple label. Recipient and subject are optional and just help you stay organized.

Note: many email clients block remote images until the recipient allows them, so an "open" only appears after the image is actually loaded.
